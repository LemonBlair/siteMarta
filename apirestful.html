API RESTful vs api REST api rest --> podemos enviar sustantivos o verbos, mandarle acciones y datos. Nombre: Juan // Apellido: López // Acción: Insertar --> lo primero que lee es la acción, y en función de eso, hace algo con los datos. api restful --> solo mandamos datos.
